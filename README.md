# Venta de Entradas - Frontend

Frontend de la plataforma de venta de entradas, desarrollado con Next.js.

## Requisitos

- Node.js 18+
- npm

## Instalación

```bash
npm install
```

## Configuración

Copia el archivo de ejemplo y configura tus variables de entorno:

```bash
cp .env.local.example .env.local
```

## Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:3000`.
